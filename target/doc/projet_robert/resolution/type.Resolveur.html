[−][src]Type Definition projet_robert::resolution::Resolveur

type Resolveur = fn(_: &mut Contexte, _: ArgumentsLocaux) -> Retour;

Un type spécifique au projet : le type 'Résolveur' est la signature d'une fonction de résolution, quelque soit le module de résolution. Elle prend deux paramètres : le contexte du socket ainsi qu'un objet permettant de récupèrer à la demande les arguments dits 'locaux' (propre à une requête). La fonction renvoie un objet "retour", qui sera transmis au client via une série d'octets écrite sur le socket. La définition de cette signature a pour principal but de soulager les signatures dans d'autres fonctions de résolution.
